# 🧠 Giteando — Cheatsheet práctico de Git

Un repo simple, directo y sin humo para **no volver a googlear comandos de Git en medio del caos**.

---

## 🚀 Objetivo

Tener a mano:

* Comandos básicos (los de todos los días)
* Flujos reales (no teoría de manual)
* Soluciones a errores típicos
* Casos concretos: GitHub, GitLab, multi-remotos

---

## ⚡ Uso rápido

Cuando algo no funciona:

1. Respirás (opcional)
2. Venís a este repo
3. Copiás y pegás
4. Seguís laburando

---

## 🧩 Casos cubiertos

* Subir carpeta local a GitHub
* Migrar de GitLab (asimov) a GitHub
* Trabajar con dos remotos
* Manejo básico de ramas
* Fixes express cuando Git se pone creativo

---

## 🔁 Filosofía

Git no es difícil.
Es consistente… pero no perdona distracciones.

Este repo no intenta explicar Git.
Intenta que **no te bloquee**.

---

## 🧨 Disclaimer

Si algo se rompe:

* probablemente hiciste un `push` donde no debías
* o un `pull` sin mirar
* o Git decidió enseñarte una lección

Este repo reduce esas probabilidades, no hace magia.

---

## 🧱 Regla de oro

Antes de cualquier cosa:

git status

---

## 🏁 TL;DR

Git no es el enemigo.
El enemigo es no tener este repo a mano.
